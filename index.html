TuitionConnect (TC)

  • This is a desktop application for private tutors. It has a GUI but most of the user interactions happen using a CLI (Command Line Interface).

  • For Tutors:

    1. Provides easy management of task with the aid of a schedule.

    2. Has a calendar that makes viewing of task easier.

    3. Able to add, delete, edit any tuition or personal task into the schedule.

    4. Easily sort and group students by certain information.

    5. Has a log book that summarises the week for you.
